# Suhas Maddi Portfolio (GitHub Pages)

Modern one-page portfolio site built with HTML + Tailwind CSS and designed for recruiter-friendly scanning.

## Files

- `index.html` - main portfolio page
- `Suhas_Guptha_Maddi.pdf` - downloadable resume
- `assets/.gitkeep` - placeholder for future images/icons

## Run locally

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Create a new GitHub repo (for example: `suhas-portfolio`).
2. Push this folder content to the repo.
3. In GitHub repo settings, go to **Pages**.
4. Set source to `main` branch and `/ (root)` folder.
5. Save and wait for deployment.

Your site URL will be:

- `https://<github-username>.github.io/suhas-portfolio/`

If you name the repo exactly `<github-username>.github.io`, your URL will be:

- `https://<github-username>.github.io/`

## Quick customization

- Replace LinkedIn placeholder URL in `index.html` with your actual profile URL.
- Add project screenshots to `assets/` and link them in the projects section.
